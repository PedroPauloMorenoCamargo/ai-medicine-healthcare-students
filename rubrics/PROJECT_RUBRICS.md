#### Medical Problem Statement (1%)
**Excellent (0.9-1.0%):**

- Clinical significance explained
- Current approaches and limitations described
- Measurable impact articulated

**Good (0.7-0.8%):**
- Clear problem statement

**Satisfactory (0.5-0.6%):**

## üë• Individual Contribution Assessment

After each major milestone (Proposal, PR1, PR2, Final), students complete:

### Peer Evaluation Form
Rate each team member (including yourself) on:
1. **Contribution to technical work** (1-5)
2. **Communication & collaboration** (1-5)
3. **Reliability & meeting deadlines** (1-5)
4. **Leadership & initiative** (1-5)

**Average score:** 4.0+ = Full credit | 3.0-3.9 = -5% | 2.0-2.9 = -10% | <2.0 = -15%

### Individual Contribution Statement
Submit with final deliverable:
- Specific tasks completed
- Challenges overcome
- Learning outcomes

### Code Commit Analysis
GitHub insights reviewed for:
- Commit frequency
- Lines of code contributed
- Code review participation

**Final individual grade = Base group grade ¬± 15% (based on above factors)**
